# E-Commerce Sales Analysis

## 📌 Project Overview
This project analyzes sales data to find trends, top products, and region-wise performance.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Analysis Done
- Total sales calculation
- Product-wise sales
- Region-wise sales
- Monthly sales trend

## 💡 Insights
- Laptop is the top-selling product
- South region has strong sales
- Sales show an increasing trend

## 📁 Files
- analysis.py (code)
- sales.csv (dataset)
